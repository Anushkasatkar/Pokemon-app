
# Pokémon Search App

This is a fun and engaging React webapp that lets you search, browse, and explore Pokémon using the [PokéAPI](https://pokeapi.co/).

## Features

- **Search Pokémon** by name
- **View Pokémon cards** in a responsive grid
- **Click a Pokémon** to see detailed info
- **View stats and types** for each Pokémon
- **See the full evolution chain**

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```
2. **Start the development server:**
	```bash
	npm run dev
	```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack
- React
- Vite
- PokéAPI
